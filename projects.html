Role Bot

Using discord.js, I developed a bot that manages users' roles in a server of 60+ people. The bot allows members of the server to create custom roles and add anyone to these roles while simultaneously blocking users from granting dangerous permissions. After writing the code, I used SSH to load the bot onto a raspberry-pi that now runs the bot 24/7 on my home network.

Golf Membership Tracker

Using C++ I developed a small command line program that modifies a ledger of data in order to track metrics such as average score per 9 holes. Most importantly the tracker keeps a running total of how much money the user has saved in using their membership. I developed this program to ensure that I was getting my money's worth out of my membership!

Pentago*

Using the popular Python library Pygame, I recreated the board game Pentago. The game is essentially an extremely intense version of tic tac toe in which four 3x3 grids are at play. Each turn players have the option to rotate any one of the grids 90 degrees instead of placing a piece. This functionality required a matrix rotation algorithm in order to keep track of each piece and their respective postions on the board after a rotation is completed. This project is still under development as I am looking into methods of deploying the game on the web. Rather than describe this project, I will soon be able to show you!
